webpack

webpack初步的搭建与配置

环境搭建

1.全局安装

cnpm i -g webpack

2.本地安装

cnpm i webpack --save-dev

3.开始打包

webpack ./main.js(主文件路径) bundle.js(出口文件路径)

配置

配置文件 webpack.config.js

1.引入webpack模块
    var webpack = require('webpack');
2.导出配置对象
    //导出配置对象
    module.exports={
        //声明入口主文件
        entry:'./js/main.js',
        //声明出口文件(最终生成的文件名)
        output:{
            //声明出口文件的路径
            path: './output/',
            //声明出口文件名
            filename:'bundle.js'
        },
        //声明模块加载器
        module:{
            //声明具体的模块加载器
            loaders:[
                //css加载器:让所有的以.css结尾的文件都使用css加载器进行解析
                {test:/\.css$/,loader:'style-loader!css-loader'},
                {test:/\.less$/,loader:'style!css!less'}
            ]
        }
    };

ajax

ajax是什么?

简单的来说ajax就是一种在不刷新页面的情况下向服务器发送请求,并接收响应数据的技术.

怎么用ajax发送请求接收数据(原生)

1)创建ajax引擎
    var ajax = new XMLHttpRequest();
2)构建请求
    ajax.open('请求方式','请求的url地址');
3)发送请求
    ajax.send();
4)监听事件接收服务器端的响应数据
    >ajax.onreadystatechange=function(){
        if(ajax.readyState==4 $$ ajax.status == 200){
            var data = ajax.responseText;
        }
    }
