React环境搭建(yeoman脚手架)

1.全局安装yeoman使用

cnpm i -g yo

2.全局安装react-webpack

cnpm i -g generator-react-webpack

3.使用yo命令创建项目

yo react-webpack

4.使用 npm start 启动项目

npm start
