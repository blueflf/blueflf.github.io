ajax是什么?

简单的来说ajax就是一种在不刷新页面的情况下向服务器发送请求,并接收响应数据的技术.

怎么用ajax发送请求接收数据(原生)

1)创建ajax引擎
    var ajax = new XMLHttpRequest();
2)构建请求
    ajax.open('请求方式','请求的url地址');
3)发送请求
    ajax.send();
4)监听事件接收服务器端的响应数据
    >ajax.onreadystatechange=function(){
        if(ajax.readyState==4 $$ ajax.status == 200){
            var data = ajax.responseText;
        }
    }
